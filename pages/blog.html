Question-1: What are the different ways to select an element in the DOM?

Question-2: What is the difference between innerHTML, innerText, and textContent ?

By using innerHTML() method we can see the inner html of an element and by using innerText() method we can see the inner text of an element and by using textContent() method we can see the inner text of an element with hidden text

Question-3: What is event delegation in the DOM?

By assigning the event to the parent node we can get the child element this is the event delegation in the DOM

Question-4: What is event bubbling in the DOM?

If we click on browser the browser don't know where it is clicked it goes like window>document>html>head>title>body>h>ul>li capturing stage when it gets the target element(li) than if there are any event given inside the capturing stage elements they will bubble like li>h>body>title>head>html>document
